## 案例1：北交所链接普通请求下载失败

- 公司：常州三协电机股份有限公司、中科星图测控技术股份有限公司
- 股票代码：920100、920116
- 环节：下载
- 问题表现：使用 urllib 或 requests 直接访问北交所原始 PDF 链接时出现 HTTP 302 重定向问题，无法稳定下载。
- 可能原因：北交所官网对普通程序化请求存在跳转限制，浏览器访问可以正常打开。
- 已尝试方法：保留原始官方 PDF 链接，不使用第三方直连 PDF 地址；在下载脚本中针对 bse.cn 链接采用 Selenium 控制 Chrome 浏览器自动化下载。
- 当前状态：已通过 selenium_original_url 方法由代码完成下载。
- 后续建议：后续批量处理北交所样本时，保留普通下载与浏览器自动化下载两套流程，并在 download_log.csv 中记录实际下载方法。
