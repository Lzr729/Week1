# 数据来源说明

## 1. 第一周公共样本来源

第一周使用8家公共训练样本，覆盖主板、创业板、科创板和北交所。样本来源以任务书公共样本为基础，并结合最新可获取的招股说明书样本进行替换和更新。

## 2. 样本调整说明

本次样本中，MB001由原天和磁材调整为三联锻造，BSE002由原大鹏工业调整为星图测控。调整原因是新样本中包含较清晰的创业投资基金股东、增资信息或发行前股东结构信息，更适合用于第一周PE/VC投资信息抽取训练。

## 3. 招股说明书来源平台

本周招股说明书主要来自以下平台：

- 巨潮资讯网
- 北京证券交易所官网
- 上海证券交易所相关披露入口
- 深圳证券交易所相关披露入口

## 4. 文件筛选原则

本项目只选择标题中明确包含“招股说明书”的文件。

以下文件不作为目标文件：

- 上市公告书
- 发行公告
- 发行结果公告
- 问询回复
- 审核问询函回复
- 提示性公告

## 5. 记录规则

每家公司需要在 company_lists/week1_public_samples.csv 中记录招股说明书标题、来源平台、来源页面链接、PDF链接、披露日期和文件版本。

如果链接失效、文件无法下载或文件类型不符合要求，需要在 notes 字段和 logs/error_cases.md 中说明。
